# **CrimeManagement**
This is a repository that manages crime.
---
## **Description**

This is a system managing crime developed in Java using Netbeans IDE. The project help police departments manage and keep up with officer assignments, case details and criminal records efficiently.

## **Installation**
You will need the following installed:
Java(TM) SE Development kit 21.0.7
Netbeans IDE 23
MYSQL Workbench 8.0 CE
JDBC Connector driver for your database
---
## **Usage**
Open the netbeans IDE go to file then open project
Open MYSQL WORKBENCH to create a new database crime_management on your local server the execute the SQL Script to create the required tables
On netbeans IDE right-click on libraries, select ADD JAR/FOLDER and add the JDBC connector JAR file 
Right-click the project name the select run for the output after writing the code
